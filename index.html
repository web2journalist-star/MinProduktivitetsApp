v0 (work in progress)

MinProduktivitetsApp

En lokal-first produktivitetsapp med audit-log (events), Neon (Postgres) og lokal LLM via Ollama. Målet er at skabe en "Hvis jeg kun har 1 time" plan der er konkret, skarp og brugbar.

Data: Neon (Postgres)
LLM: Ollama (lokalt)
Log: events (audit)
Scope: MVP i små, sikre trin

Hvad er klar allerede?

Backend runner (lokalt)
Scriptet app/backend/llm_plan.py henter dagens fokus + to-do’s fra Neon, kalder Ollama og logger svaret som LLM_PLAN_GENERATED.
Docs
Se projektets docs for vision, data model, token budget og working agreement.

Kør lokalt

cd app/backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python llm_plan.py
Kræver at du har sat DATABASE_URL i .env og at Ollama kører lokalt.

Vercel note

Denne side er deployet som statisk output. Selve LLM-kørslen foregår lokalt (Ollama). Et næste skridt er at deploye en lille web-UI/API der kan læse fra Neon (uden Ollama), eller skifte til en cloud-LLM for fuld online oplevelse.

Bygget iterativt med fokus på sikkerhed, small steps, og audit-log.
