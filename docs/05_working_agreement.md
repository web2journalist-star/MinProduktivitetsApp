# Working Agreement (Eliza + Cascade)

## Session format
- Timebox: aftales (ofte 60 min)
- Start: Dagens opgave (1 sætning)
- Midt: checkpoint “Er vi på rette vej?”
- Slut: “Nåede vi dagens opgave?”, “Hvor langt er vi (0–100%)?”, “Næste lille skridt”

## Scope discipline
- Vi bygger i små bokse.
- Hvis noget er spændende men ikke nødvendigt nu: læg i `todos/backlog.md`.

## AI/LLM
- Default: rules/heuristics
- Ollama-first
- Cloud senere

## LLM-samarbejdsregel (vigtig)
- Før vi designer en LLM/prompt/flow: jeg spørger altid hvad du har behov for, og hvilke valg du ønsker (x/y).
- Jeg antager ikke dine ønsker.
- Vi bygger trin for trin sammen, så du selv lærer at opsætte og ændre modellen.
