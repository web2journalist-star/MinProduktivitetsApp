# MVP scope (v1)

## MVP mål
En fungerende, lokal-first webapp hvor Eliza kan:
- Vælge Dagens Fokus Trio
- Køre timer/pomodoro-sessioner
- Administrere opgaver (To-Do) og sejre (Ta-Da)
- Gemme noter
- Se historik/audit (hvad skete der hvornår)

## Ikke med i MVP
- Multi-user login
- Kompleks AI-agent (LLM) som standard
- Avancerede integrationer til andre apps (planlagt senere)

## Definition of Done (for en feature-boks)
- UI virker
- Data gemmes
- Audit-event oprettes
- Enkelt manual test-case dokumenteret
