# Roadmap (bokse)

- Projektstruktur + docs + db/schema

## Boks 3 — Backend API (senere)
- CRUD endpoints
- Rules endpoint (uden LLM)

## Boks 4 — Frontend (React læringsspor)
- UI-komponenter
- API integration

## Boks 5 — Local LLM (Ollama)
